# Guthub Pages

- webページとして公開したいリポジトリ（公開設定をpublicにしておく）で、Settings -> GitHub Pages -> branchを選択してsave。
- URLは`https://[user_name].github.io/[repository_name]/`となる。
- リポジトリが更新されるとwebページも自動的に更新される。
